Códigos de productos
====================

Dispone de un nueva pestaña en la ficha del producto para añadir códigos de productos

* code39
* ean
* ean13
* ean8
* gs1
* gtin
* isbn
* isbn10
* isbn13
* issn
* jan
* pzn
* upc
* upca
